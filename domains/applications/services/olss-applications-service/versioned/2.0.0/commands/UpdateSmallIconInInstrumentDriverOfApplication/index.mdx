## Overview
[NOT IMPLEMENTED] This method is used to create or update the small icon for the specified instrument driver.
A calling user will need to have been granted Controller_Edit permission.

##### Use Cases

1. At upgrade, an improved icon is registered.





## PUT `(/v1.0/applications/{appid}/drivers/{id}/smallicon)`

### Parameters
- **appid** (path) (required): It is either Application's id or unique key.
- **id** (path) (required): Identifier of an object
- **Accept-Language** (header): The locale for any localized string returned by the method.

Supported locales are:
* en-US
* ja-JP
* pt-BR
* ru-RU
* zh-Hans

Default is the server locale.
